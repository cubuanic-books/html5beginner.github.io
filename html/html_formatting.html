Элементы форматирования в HTML

В предыдущем уроке вы узнали об HTML атрибуте style. HTML также определяет специальные элементы для придания тексту особого значения. HTML использует такие элементы, как <b> и <i> для форматирования вывода, например жирний или курсивный текст. Элементы форматирования предназначены для отображения специальных типов текста.

  • <b> - жирный текст
  • <strong> - важный текст (семантически важный, выводится жирным)
  • <i> - курсивный текст
  • <em> - подчёркнутый текст (семантически важный, выводится курсивом)
  • <mark> - помеченный (маркированный) текст
  • <small> - уменьшенный текст
  • <del> - удаленный текст
  • <ins> - вставленный текст
  • <sub> - текст подстрочный (нижний индекс)
  • <sup> - текст надстрочный (верхний индекс)

HTML элементы <b> и <strong>

HTML элемент <b> определяет жирный текст без дополнительного значения.

Пример:

<b>This text is bold</b>

HTML элемент <strong> определяет важный текст с добавлением семантически «сильного» значения.

Пример:

<strong>This text is strong</strong>

HTML элементы <i> и <em>

HTML элемент <i> определяет текст курсивом без дополнительного значения.

Пример:

<i>This text is italic</i>
Попробуйте сами »

HTML элемент <em> определяет подчёркнутый текст с дополнительным смысловым значением.

Пример:

<em>This text is emphasized</em>
Попробуйте сами »

Примечание: Браузеры отображают <strong> как <b> (выделяют жирным шрифтом), а <em> как <i> (выделяют курсивным шрифтом). Однако, существует разница в значении этих тегов: <b> и <i> лишь отображаются жирным и курсивным текстом, но теги <strong> и <em> означают, что текст "важный" по значению.

HTML элемент <small>

HTML элемент <small> определяет меньший текст (по сравнению с текстом по умолчанию):

Пример:

<h2>HTML <small>Small</small> Formatting</h2>

HTML элемент <del>

HTML элемент <del> определяет удалённый (изъятый) текст. По умолчанию отображается зачёркнутым.

Пример:

<p>My favorite color is <del>blue</del> red.</p>

HTML элемент <ins>

HTML элемент <ins> определяет вставленный (добавленный) текст. По умолчанию отображается подчёркнутым.

Пример:

<p>My favorite <ins>color</ins> is red.</p>

Пример:
