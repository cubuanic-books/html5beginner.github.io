HTML расшифровывается как Hyper Text Markup Language (язык гипертекстовой разметки)

HTML стандартный язык разметки веб-страниц

HTML элементы являются строительными блоками HTML страниц

HTML элементы представлены <> тегами

Примечание. HTML - НЕ ЯВЛЯЕТСЯ языком программирования!!!

HTML Элементы

HTML элемент представляет собой тег начальный (открывающий) и тег конечный (закрывающий) из содержимым между ними:

<h1>Это Заголовок</h1>
Начальный тег Содержание элемента Конечный тег
<h1> Это Заголовок </h1>
<p> Это параграф. </p>

HTML Атрибуты

  • HTML элементы могут иметь атрибуты
  • Атрибуты предоставляют дополнительную информацию об элементе
  • Атрибуты входят в пары имя/значение, такие как charset="utf-8"

Простой HTML документ

<!DOCTYPE html>
<html lang="en">

<meta charset="utf-8">
<title>Название страницы</title>

<body>
   <h1>Это заголовок</h1>
   <p>Это параграф.</p>
   <p>Это другой параграф.</p>
</body>

</html>

Объяснение примера

HTML-элементы являются строительными блоками HTML-страниц.

  • Объявление <!DOCTYPE html> определяет этот документ как HTML5
  • Элемент <html> является корневым элементом HTML страницы
  • Атрибут lang определяет язык документа
  • Элемент <meta> содержит метаинформацию о документе
  • Атрибут charset определяет набор символов, используемый в документе
  • Элемент <title> определяет заголовок документа
  • Элемент <body> содержит видимое содержимое страницы
  • Элемент <h1> определяет большой заголовок
  • Элемент <p> определяет параграф

HTML Документы

Все HTML документы должны начинаться с объявления типа документа: <!DOCTYPE html>.

Сам HTML-документ начинается тегом <html> и заканчивается тегом </html>.

Видимая часть HTML документа находится между <body> и </body>.

Структура HTML документа

Ниже приведена визуализация HTML-документа (HTML-страницы):

<html>
<head>
<title>Page title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
</body>
</html>

Примечание: Только содержимое внутри раздела <body> (белая область) отображается в браузере.

HTML Заголовки

HTML-заголовки определяются с помощью тегов от <h1> до <h6>.

<h1> определяет наиболее важный заголовок. <h6> определяет наименее важный заголовок:

Пример

<h1>Это заголовок 1</h1>
<h2>Это заголовок 2</h2>
<h3>Это заголовок 3</h3>

HTML списки определяются с помощью тегов <a>:

Пример

<a href="https://www.w3schools.com">This is a link</a>

HTML Изображения

HTML изображения определяются с помощью тегов <img>.

Исходный файл (src), альтернативный текст (alt), ширина width и высота height указываются в качестве атрибутов:

Пример

<img src="img_w3schools.jpg" alt="W3Schools" style="width:120px;height:150px"

HTML Списки

HTML списки определяются с помощью тегов <ul> (неупорядоченный/маркерный список) или <ol> (упорядоченный/нумерованный список), затем следуют теги <li> (элементы списка):

Пример

<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

HTML Таблицы

HTML таблицы определяются с помощью тега <table>.

Строки таблицы определяются с помощью тега <tr>.

Заголовки таблицы определяются с помощью тегов <th> (жирные и центрированные по умолчанию).

Ячейки таблицы (данные) определяются с помощью тегов <td>.

Пример

<table>
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
</table>

HTML программирование

Каждый HTML элемент может иметь атрибуты.

Для веб-разработки и программирования наиболее важными атрибутами являются id и class. Эти атрибуты часто используются для обработки программных веб-страниц.

Атрибут Пример
id <table id="table01"
class <p class="normal">
style <p style="font-size:16px">
data- <div data-id="500">
onclick <input onclick="myFunction()">
onmouseover <a onmouseover="this.setAttribute('style','color:red')">
