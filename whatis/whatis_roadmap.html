Front-End Roadmap

Дорожная карта фронтэнд-разработчика

ШАГ 1
Изучите основы

Каждый веб-разработчик должен иметь общее представление об HTML, CSS и JavaScript.

Адаптивный веб-дизайн используется во всех типах современной веб-разработки.

ECMAScript 5 (JavaScript 5) поддерживается во всех современных браузерах. Внимательно изучите данную тему, особенно новые функции массива.

CSS

JavaScript

JavaScript ECMAScript 5

ШАГ 2
Копать глубже

Когда вы изучите HTML и CSS, начинайте копать глубже.

Вы должны научиться использовать Карты, Шрифты и Иконки в HTML.

На стороне JavaScript вы должны узнать, как получить доступ к HTML DOM.

Вы также должны научиться использовать AJAX и JSON для выполнения запросов к серверу.

HTML DOM Google карты Google шрифты Google схемы

CSS

CSS Иконки

JavaScript

XML JSON AJAX

ШАГ 3
Выберите Framework

Теперь пришло время взглянуть на некоторые фреймворки (Framework).

На стороне CSS вы должны выбрать платформу для адаптивного веб-дизайна:
Bootstrap  /  Material Design  /  W3.CSS

На стороне JavaScript вы должны изучить хотя бы один современный framework:
React.js  /  Angular.js  /  Vue.js   /  W3.JS

Возможно, популярность jQuery достигла предела, но это всё ещё самый используемый фреймворк JavaScript при создании сайтов.
