For - перевод с анг.: Для (за).

PHP циклы for выполняют блок кода указанное количество раз.

PHP цикл for

Цикл for используется, когда вы заранее знаете, сколько раз должен выполняться скрипт.

Синтаксис

for (счетчик инициализации; счетчик теста; счетчик увеличений) {
    код для выполнения;
}

Параметры:

  • счётчик инициализации: Инициализирует значение счетчика цикла.
  • счётчик теста: Оценивается для каждой итерации цикла. Если он оценивается как TRUE, цикл продолжается. Если он оценивается как FALSE, цикл заканчивается.
  • счётчик увеличения: Увеличивает значение счетчика цикла.

В приведенном ниже примере отображаются цифры от 0 до 10:

Пример

<?php
for ($x = 0; $x <= 10; $x++) {
    echo "The number is: $x <br>";
}
?>

PHP цикл foreach

Цикл foreach работает только для массивов и используется для циклического прохождения каждой пары ключ/значение в массиве.

Синтаксис

foreach ($массив as $значение) {
    код для выполнения;
}

Для каждой итерации (повтора) цикла значение текущего элемента массива присваивается $value (значение), а указатель массива перемещается на единицу, пока не достигнет последнего элемента массива.

В следующем примере демонстрируется цикл, который будет выводить значения данного массива ($colors):

Пример

<?php
$colors = array("red", "green", "blue", "yellow");

foreach ($colors as $value) {
    echo "$value <br>";
}
?>
