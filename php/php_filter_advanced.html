Проверка целого числа в диапазоне

В следующем примере функция filter_var() используется для проверки, имеет ли переменная тип INT (целое число) и находится ли в диапазоне от 1 до 200:

Пример

<?php
$int = 122;
$min = 1;
$max = 200;

if (filter_var($int, FILTER_VALIDATE_INT, array("options" => array("min_range"=>$min, "max_range"=>$max))) === false) {
    echo("Значение переменной находится вне допустимого диапазона");
} else {
    echo("Значение переменной находится в допустимых пределах");
}
?>

Проверка IPv6 адреса

В следующем примере функция filter_var() используется для проверки, является ли переменная $ip действительным адресом IPv6:

Пример

<?php
$ip = "2001:0db8:85a3:08d3:1319:8a2e:0370:7334";

if (!filter_var($ip, FILTER_VALIDATE_IP, FILTER_FLAG_IPV6) === false) {
    echo("$ip является действительным адресом IPv6");
} else {
    echo("$ip не является действительным адресом IPv6");
}
?>

Проверка URL - должен содержать QueryString

В следующем примере функция filter_var() используется для проверки, является ли переменная $url URL-адресом со строкой запроса:

Пример

<?php
$url = "https://www.w3schools.com";

if (!filter_var($url, FILTER_VALIDATE_URL, FILTER_FLAG_QUERY_REQUIRED) === false) {
    echo("$url действительный URL");
} else {
    echo("$url не действительный URL");
}
?>
