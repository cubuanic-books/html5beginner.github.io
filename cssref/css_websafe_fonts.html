Часто используемые комбинации шрифтов

Свойство font-family должно содержать несколько названий шрифтов в качестве "резервной" системы, чтобы обеспечить максимальную совместимость между браузерами/операционными системами. Если браузер не поддерживает первый указанный шрифт, он пытается использовать следующий указанный шрифт.

Начните с нужного шрифта и завершите общим шрифтом, чтобы браузер мог выбрать аналогичный шрифт в общем семействе шрифтов, если другие шрифты недоступны:

Пример

p {
  font-family: "Times New Roman", Times, serif;
}
